# AddClassy.JS

The jQuery plugin for Matt.

## Usage: 


#### 1. Include `addclassy.jquery.js` on your page

```
<script src="addclassy.jquery.js"></script>
```



#### 2. Call jQuery's regular <strong>addClass()</strong> on an element: 

```
$('div').addClass('example'); 
```
